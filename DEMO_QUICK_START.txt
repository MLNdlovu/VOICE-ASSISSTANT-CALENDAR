=== TROUBLESHOOTING ===

If app won't start:
  - Check: pip install flask google-auth-oauthlib google-oauth google-api-python-client
  - Run: python web_app.py

If login fails:
  - Verify: .config/client_secret_*.json exists
  - Check: OAUTHLIB_INSECURE_TRANSPORT=1 is set

If voice doesn't work:
  - Check browser console for errors (F12)
  - Verify microphone permissions
  - Try Chrome/Edge (better Web Speech API support)

If events don't save:
  - Check: Access token in session
  - Verify: Google Calendar API is enabled in Google Cloud Console
